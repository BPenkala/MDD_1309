About SearchAGram

SearchAGram was developed for educational purposes only for the Web Design & Development program at Full Sail University. The goal of this was to develop an Instagram hashtag search quickly and easily with a minimalist UI.

SearchAGram allows the easy and fast browsing of Instagram photos by simply entering a hashtag into the search field.

SearchAGram was written using:

  • AngularJS
  • Twitter Bootstrap
  • HTML5
  • CSS3
  • Instagram API
